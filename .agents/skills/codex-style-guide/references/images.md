# Images

Source: <https://doc.wikimedia.org/codex/latest/style-guide/images.html>.

Photos and other media used as content. Selection guidelines:

- **Contextually relevant** — directly relate to the subject.
- **Neutral point of view** — e.g. a side profile of a shark, not an
  action/hunting shot.
- **Most accurate medium** — a photograph of a real person; a cartoon
  still for a cartoon character.
- **Good resolution** — high quality, never pixelated.
- **Respects copyright** — Creative Commons / Fair use / Public Domain,
  correctly attributed (prefer Wikimedia Commons).
