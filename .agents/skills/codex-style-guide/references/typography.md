# Typography (readability)

Source:
<https://doc.wikimedia.org/codex/latest/style-guide/typography.html>.

The core rule — **every piece of text uses one of the 9 canonical
styles** — has its own skill:
[`codex-typography`](../../codex-typography/SKILL.md). Following it gives
you the right font family and line-height automatically.

This page covers only what the styles **don't** decide:

- **Line length** — keep prose to ~**≤ 75 characters** per line for
  comfortable reading.
- **Dynamic text** — text length varies wildly across languages. Don't
  design to a fixed length: keep the UI uncrowded, let containers expand,
  shrink long content, and clip with an ellipsis only when no information
  is lost (see [`content-overflow.md`](content-overflow.md)).

For **contrast** (4.5:1), see [`accessibility.md`](accessibility.md) —
that's a colour concern, not a type-style one.
