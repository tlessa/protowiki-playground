# All Codex icons

The complete catalogue of importable icon constants from
`@wikimedia/codex-icons` (286 icons). Import any of these and
render through `CdxIcon` — see [`../SKILL.md`](../SKILL.md) for usage,
accessibility, and direction/lang handling.

The **Name** column is a readable label derived from the constant. The
**Direction** column flags icons that are not plain strings: _flips in RTL_
(auto-mirrored), _distinct LTR/RTL glyphs_, or _varies by `langCode`_.

This file is generated from the installed package. To regenerate it (e.g.
after a Codex upgrade) see the
[`protowiki-update-codex`](../../protowiki-update-codex/SKILL.md) skill.
