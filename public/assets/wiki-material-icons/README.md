# Wiki Material icon assets

Place imported SVG files from:

- https://github.com/wikimedia/apps-android-wikipedia/tree/main/icon-svgs

The icon set toggle in settings looks for these files:

- ic_home_white_24dp.svg
- ic_bookmark_border_white_24dp.svg
- ic_search_white_24dp.svg
- ic_edit_white_24dp.svg
- ic_more_vert_white_24dp.svg
- ic_notifications_white_24dp.svg
- ic_filter_list_white_24dp.svg
- ic_delete_white_24dp.svg

If a file is missing, the app automatically falls back to the Codex icon for that glyph.
